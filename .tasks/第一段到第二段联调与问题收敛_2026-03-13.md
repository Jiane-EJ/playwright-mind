## 联调建议流程

1. `npm run stage1:run:headed`
2. `npm run stage1:promote -- --requestId <requestId> --target specs/tasks/acceptance-task.generated.json --force`
3. 将 `.env` 中 `STAGE2_TASK_FILE` 指向目标任务文件
4. `npm run stage2:run:headed`
5. 检查 `result.json`、`ai_run`、`ai_run_step`、`ai_artifact` 是否完整
